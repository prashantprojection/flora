# Flora - Flutter Mobile App

A mobile plant care companion built with Flutter.

## Download the App

You can download the latest APK directly from our GitHub releases page: [Download Flora v1.1.0 APK](https://github.com/prashantprojection/flora/releases/download/v1.1.0/app-release.apk)

## Overview

Flora is a smart plant care application that helps you keep your plants healthy and thriving. This Flutter application provides an intuitive mobile experience for managing your indoor garden.

## New in v1.1.0 (Production Release)

*   **Premium Dashboard UI:** A completely redesigned home screen with dynamic time-based greetings ("Good Morning/Evening"), stunning gradients, and a masonry grid layout for your plant collection.
*   **Visual Schedule:** Task cards now display your actual plant photos, making it easier to identify what needs care at a glance.
*   **Restored Quick Actions:** The "Add Plant" floating action button is back for instant access.
*   **Smart Health Check:** Diagnose plant issues by taking a picture.
*   **Stability Improvements:** Fixed layout overflows and deprecated API usage for a smoother experience.

## Getting Started

### Prerequisites

*   Flutter SDK (version 3.x.x or higher)
*   An editor like VS Code or Android Studio with the Flutter plugin.

### Installation

If you want to build the app from source, clone the repository from GitHub:

1.  Navigate to the `flora` directory:
    ```bash
    cd flora
    ```
2.  Install dependencies:
    ```bash
    flutter pub get
    ```
3.  Run the application:
    ```bash
    flutter run
    ```
