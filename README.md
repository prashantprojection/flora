# Flora - Flutter Mobile App

A mobile plant care companion built with Flutter.

## Download the App

You can download the latest APK directly from our GitHub releases page: [Download Flora v1.1.0 APK](https://github.com/prashantprojection/flora/releases/download/v1.1.0/app-release.apk)

## Overview

Flora is a smart plant care application that helps you keep your plants healthy and thriving. This Flutter application provides an intuitive mobile experience for managing your indoor garden.

## Features

*   **Plant Dashboard:** View all your plants at a glance.
*   **Plant Details:** See detailed information about each plant, including care history.
*   **AI-Powered Health Check:** Diagnose plant issues by taking a picture.
*   **AI-Generated Care Tips:** Get personalized care recommendations for your plants.
*   **Care Schedule:** Keep track of your plant care routine.

## Getting Started

### Prerequisites

*   Flutter SDK (version 3.x.x or higher)
*   An editor like VS Code or Android Studio with the Flutter plugin.

### Installation

If you want to build the app from source, clone the repository from GitHub:

1.  Navigate to the `flora` directory:
    ```bash
    cd flora
    ```
2.  Install dependencies:
    ```bash
    flutter pub get
    ```
3.  Run the application:
    ```bash
    flutter run
    ```
